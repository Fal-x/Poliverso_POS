# Manual Operativo UI

## Inicio de turno
1. Seleccionar usuario.
2. Ingresar PIN.
3. Abrir caja con conteo físico.

## Ventas
- Seleccionar productos.
- Confirmar medio de pago.
- Cobrar.
- Validación backend obligatoria.

## Recargas
- Leer tarjeta.
- Ingresar monto.
- Confirmar pago.
- Refrescar saldo desde backend.

## Cierre de caja
- Revisar efectivo esperado.
- Contar efectivo físico.
- Confirmar cierre.
- Si hay diferencia: pedir autorización.

## Autorizaciones
- Siempre requieren PIN supervisor.
- Motivo obligatorio.
